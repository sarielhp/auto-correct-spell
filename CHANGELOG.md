# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-05-16

### Added
- Initial release of `auto-correct-spell`.
- Integration with Jinx spellchecker for automatic abbrev creation.
- Global minor mode `auto-correct-spell-mode`.
- Command `auto-correct-spell-remove-last-abbrev` to undo the most recent correction.
- Word filtering based on length and alphabetic characters.
- Persistence of abbrevs to the user's abbrev file.
- Comprehensive test suite.
- GPLv3 license.
